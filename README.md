# **PROJETO WEB**

### *Simula um site onde é possivel nagevar entre as paginas atraves de botoões*

## **detalhes importantes**

Apenas um pequeno projeto mostrando conhecimento e organização em um codigo de HTML e CSS, utulizei botões para a navegação entre páginas então tem botões na parte superior e inferior do site

## Ferramentas utilizadas

- `CSS`
